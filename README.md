## 🚀 Instalasi & Menjalankan

1. **Clone repository:**
   ```bash
   git clone https://github.com/ardlikafi/sentient-chess.git
   cd sentient-chess
   ```
2. **Install dependencies:**
   ```bash
   flutter pub get
   ```
3. **Jalankan aplikasi:**
   ```bash
   flutter run
   ```
